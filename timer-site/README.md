## ChronoCanvas — Mindful Timer

ChronoCanvas is a single-page experience for designing your focus rituals. It blends a design-forward countdown timer, curated presets, and a lightweight session journal to help you move through focus, reset, and reflection cycles with intention.

### Features

- Immersive radial timer with live progress, gradient lighting, and responsive layout
- Quick presets for Pomodoro, deep focus, and short creative sprints
- Custom minute slider and numeric input with 1–90 minute range
- Session log that records pauses, completions, and preset loads
- Contextual rituals to swap durations on the fly
- Built with Next.js App Router, Tailwind CSS, and optimized for deployment to Vercel

### Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to explore the timer.

### Production Build

```bash
npm run build
npm start
```

Deploy seamlessly on Vercel with `vercel deploy --prod`.
